# Projet de régression logistique

Projet réalisé en **Licence 3 Mathématiques-Informatique**.

L’objectif est d’implémenter en Python une **régression logistique depuis zéro**, sans utiliser de bibliothèque de machine learning .

## Contenu

* `regression_logistique_ouedraogo_wendyida.ipynb` : notebook Jupyter avec le code, les explications, les graphiques et les résultats.
* `rapport_regression_logistique_ouedraogo.pdf` : rapport du projet.

## Méthode

Le notebook contient :

* création du jeu de données ;
* séparation entraînement/test ;
* normalisation ;
* fonction sigmoïde ;
* log-loss ;
* descente de gradient ;
* prédiction ;
* matrice de confusion ;
* accuracy, précision, rappel et F1-score ;
* interprétation des coefficients.
